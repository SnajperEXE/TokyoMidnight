# $${\color{lightgreen}TokyoMidnight - Mint \space Edition \space (WIP)}$$

A modern Discord theme featuring mint neon accents, polished midnight tones, and platform-inspired link styling.

<img width="1920" height="1075" alt="tokyo-midnight-screenshotmain" src="https://github.com/user-attachments/assets/9ed158db-f5df-4371-811e-75f3b5191911" />

## ✨ $${\color{red}Features}$$

- Midnight-inspired dark aesthetic
- Consistent mint neon design language
- Enhanced embeds, media controls, and chat elements
- Platform-colored links and integrations
- Subtle animations and glow effects
- Custom date markers and message indicators
- Expandable Member List
