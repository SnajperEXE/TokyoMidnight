# TokyoMidnight - Mint Edition

A modern Discord theme featuring mint neon accents, polished midnight tones, and platform-inspired link styling.

---

## ✨ Features

- Mint neon aesthetic
- Platform-colored links
- TikTok RGB/glitch effect
- Steam gradient links
- Modern dark UI
- Soft glow effects
- Custom embed styling
